# ICONS GENERATOR

## Steps to run

1. Configure .env
2. `yarn`
3. `yarn build-icons-script`
   <!-- Pulls figma icons and builds components -->
   <!-- ## If you ever need to edit the generation script (generate.ts) then you should run `yarn build-icons-script` afterwards -->
4. `yarn icons`
5. `yarn build`
